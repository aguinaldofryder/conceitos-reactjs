# Conceitos ReactJS
Projeto para praticar alguns conceitos do ReactJS, como o uso de hooks, constrole de estado da aplicação e chamadas a uma API Rest.

## Tecnologias
* ReactJS
* Axios
* Babel
* Webpack

## Instalação
```
git clone https://github.com/aguinaldofryder/conceitos-reactjs.git
cd conceitos-reactjs
yarn install
yarn start
```
## Testes
```
yarn test
```
